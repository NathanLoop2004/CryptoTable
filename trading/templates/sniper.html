🚨 Flags de Peligro (DANGER)

🍯 Honeypot

No puedes vender el token. Compras pero nunca puedes recuperar tu dinero.

🖨️ Mintable

El dueño puede crear tokens nuevos de la nada, causando inflación y bajando el precio.

🚫 Blacklist

El dueño puede bloquear wallets específicas para que no puedan vender ni transferir.

⏸️ Pausable

El dueño puede pausar todo el trading cuando quiera (ej: pausa ventas justo cuando sube).

🔐 Can't sell

El contrato impide vender todos tus tokens — te quedarás con tokens atrapados.

⚠️ Ctrl balance

El dueño puede cambiar el balance de cualquier wallet directamente. Puede quitarte tokens.

💣 Self-destruct

El contrato puede autodestruirse, desapareciendo junto con tu inversión.

⚠️ Flags de Advertencia (WARNING)

🔄 Proxy

El contrato puede ser actualizado. El dueño podría cambiar toda la lógica después de que compres.

👤 Hidden owner

El dueño existe pero está oculto. No puedes ver quién controla el contrato.

👑 Has owner

El contrato tiene un dueño con privilegios especiales (puede cambiar fees, etc.).

🔒 Not verified

El código fuente no está publicado. No se puede auditar qué hace el contrato realmente.

✅ Flags Positivos (SAFE)

✅ Safe

No se encontraron flags peligrosos. El contrato parece seguro (nunca 100% garantizado).

📝 Verified

El código fuente está publicado y verificado en la blockchain. Cualquiera puede auditarlo.

🦎 CoinGecko

El token está listado en CoinGecko — señal de legitimidad fuerte.

🌐 Website

El token tiene página web registrada (DexScreener). Señal positiva.

📱 Socials

El token tiene redes sociales (Twitter, Telegram, etc). Señal positiva.

✅ TS 60+

TokenSniffer score ≥ 60/100 — análisis automático de código positivo.

🎯 Bienvenido al Sniper Bot

🔗

1. Conecta tu Wallet

Conecta Trust Wallet o MetaMask haciendo click en el botón de wallet arriba a la derecha. Necesitas tener BNB/ETH para hacer trades.

⚙️

2. Configura los Settings

Elige tu perfil (Novato = seguro, Avanzado = agresivo). Ajusta la liquidez mínima, buy amount y take profit/stop loss. Los tooltips ⓘ te explican cada opción.

▶️

3. Inicia el Bot

Selecciona la cadena (BSC o ETH) y pulsa Start. El bot escaneará nuevos tokens en tiempo real a través del pipeline de análisis.

🔍

4. Revisa los Tokens

Los tokens detectados aparecen en Detectados. Verás el riesgo, taxes, liquidez y flags de seguridad. Solo los tokens que pasan TODOS los filtros aparecen como oportunidad de snipe.

💰

5. Ejecuta Trades

Cuando aparece una Snipe Opportunity, puedes comprar manualmente o activar Auto-Buy. Tus posiciones se muestran en la tabla de posiciones activas.

📊

6. Monitorea el Dashboard

El Performance Dashboard muestra tu P&L, win rate, velocidad de detección y efectividad del bot en tiempo real. Configura alertas de Telegram/Discord para no perder oportunidades.
